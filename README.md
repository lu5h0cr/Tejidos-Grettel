# Yelin Boutique

Página web estática para vender bolsos tejidos.

## Archivos principales

- `index.html`: estructura de la página.
- `assets/styles.css`: diseño visual, colores, fondo floral y responsive.
- `assets/script.js`: función para ampliar imágenes y mostrar descripción.
- `assets/img/`: imágenes de los bolsos.

## Cómo subirlo a GitHub Pages

1. Crear un repositorio nuevo en GitHub, por ejemplo: `yelin-boutique`.
2. Subir todos los archivos de esta carpeta al repositorio.
3. Ir a **Settings > Pages**.
4. En **Build and deployment**, elegir:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guardar y esperar el enlace público.

## Importante

En `index.html` y `assets/script.js` aparece el número `50600000000`. Cambialo por el número real de WhatsApp de Yelin Boutique.
